The ledger,
as a sentence.

Consolidated RN/Expo (RN Web) app for double-entry journal capture, ledger review, and financial statements. Built entirely on the Forest Reserve tokens. Every monetary figure is decimal-exact, right-aligned, tabular. The defining move: a balanced entry reads as a sentence, never a spreadsheet — and 借方=貸方 is shown, enforced, and impossible to break before it is shown, enforced, and impossible to break in the DB.

Platforms

One codebase. Mobile = bottom tabs (RN Paper), Web = top nav (RN Web).

Money

NUMERIC · decimal-exact · tabular · slashed-zero. No float drift, ever.

Auth

Keycloak OIDC SSO. Rows scoped to the signed-in owner.

01

Tokens — inherited

No new color or type. This app reads entirely from the brand system; the only additions are state aliases mapped onto existing semantics.

Semantic mapping
--debit → 借方 · outflow · over
--credit → 貸方 · inflow · under
--accent → primary CTA on warm surfaces
--highlight → focus ring · active row
Rules
  • Numbers: always mono, tabular, slashed-zero, right-aligned.
  • Sentences & headings: serif. UI chrome & data: mono.
  • No pill shapes — radii sm/md only. This is a ledger.
  • Color only on debit, credit, accent, focus. Surfaces stay paper.
